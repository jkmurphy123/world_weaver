# world_weaver

Milestone 1 scaffold for the fictional newsroom simulator.
Milestone 2 adds initial world generation via `newsroom init-world`.

## Run

```bash
cd /home/ubuntu/ai_projects/world_weaver
source .venv/bin/activate
pip install -e '.[dev]'
newsroom serve
newsroom init-world --prompt "A synthetic island city ruled by corporate blocs."
```
